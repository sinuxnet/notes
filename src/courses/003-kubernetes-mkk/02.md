# 2. Installing Kubernetes

## 2.1 Installing Methods

* Educational: `Minikube`, `KOps`, `Kubespray`, `kind`

* Production:
  * On-premises:`kubeadm`
  * Managed: GCP, AWS, Azure, ...

## 2.1 Installing Kubernetes Using `kind`

[kind](https://sigs.k8s.io/kind) is a tool for running local Kubernetes clusters using Docker container “nodes”.
kind was primarily designed for testing Kubernetes itself, but may be used for local development or CI.

### `kubectl`

Kubernetes provides a command line tool for communicating with a Kubernetes cluster's control plane, using the Kubernetes API.

This tool is named `kubectl`.

## 2.2 Installing Kubernetes Using `minikube`

`minikube` is local Kubernetes, focusing on making it easy to learn and develop for Kubernetes.

```shell
minikube start
```
